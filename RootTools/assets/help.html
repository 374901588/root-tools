★软件介绍
这是一款用于 Android 系统的的管理工具，它可以方便你对 Android 手机作出各种操作。在很多情况下，手机有了 Root 权限后，用户并不知道自己多了什么权利，可以做些什么平时做不到的，本软件即是针对 Root 后的手机，让用户享受 Root 带来的便利与快乐。
★权限说明
本软件需要三种权限，分别是 1)读写 SD 卡内容，2)访问互联网，3)获取手机状态信息，它们分别被用于 1)在 SD 卡上写入配置或缓存文件，在 SD 卡上保存用户需要备份的资料，2)从网络上检查更新，从网络上获取所需的数据，3)获取手机的状态信息，以便程序能判断手机型号，配置等，按不同的手机状态为用户提供更好的服务。本软件不会收集用户信息，请放心使用。
★运行条件
要正常使用本软件，您的手机需要有 Root 权限并且正确安装了 busybox，这在一般的第三方 ROM 中都已集成，若是没有集成 busybox 或是 busybox 版本过旧，本软件也可以为用户安装上正确的 busybox。未安装 busybox 的情况下，将会有部分功能无法使用，那些功能会有警告图标标识出来。本软件需要运行在 Anroid 2.2 或更高版本，对于 HTC Legend(G6) / HTC Desire(G7) 或是一些特定的机型，使用本软件需要将手机解为 S-OFF 状态。在较新款的手机如 HTC Evo 3D / HTC One X 等，使用官方解锁则不需要再解 S-OFF。
★增加/删除系统应用
在取得 Root 权限后，可以删除系统内的应用程序，本功能可以快捷的删除系统内置的程序，并将关联的数据一起清理干净，保证了系统内无垃圾文件残留。在设置中可以选择删除前是否要备份，以防误删了程序使系统产生问题。本项功能也允许用户将应用程序安装到系统目录内，使其无法被常规手段删除，如果误删了系统文件，也可以用安装程序到系统的方法，将备份的程序重新装回。
★备份/还原应用数据
本功能专为刷机党提供，在刷机前可以先对程序进行备份，包括应用程序的数据也可一同备份。只需选中要备份的应用，再点击“备份”按钮即可，支持同时备份多个应用，备份完成后会告知操作情况，备份成功或失败。同样的操作也可用于恢复数据，在恢复时，若程序需要被安装到 SD 卡内，可能会提示恢复失败，此时其实已经恢复成功，只是系统返回的数据并未反映真实情况。
★禁用/启用应用内组件
组件管理是一个非常有特色的功能，用户经常会遇到这样的场景，例如看微博时不愿看到广告条，本功能即可以为你实现这个需求。你只需在组件管理页面中，找到指定的应用程序，点击看到它所拥有的全部内部组件，长按即可将指定的条目禁用。若是你禁用了广告的条目，那么就不会再显示广告。更多可禁用的可以自行摸索，若是误禁用了某一条目，同样只需长按就可将其恢复成可用状态。
★清理内存
Android 手机经常越用越慢，即是因为启动的程序太多，将内存消耗殆尽，常规的清理内存的软件均是通过杀掉进程的方式来清理，而本软件是通过系统内部的机制来完成，不杀进程也可以快速的还你一个流畅的系统，当然你也可以在设置里指定清理时要杀掉进程，效果会更好。
★修改 Hosts
本功能是针对国内的网络条件而研发的，它可以帮助你搜索域名对应的 IP 地址，从而一定程度上绕过屏蔽。需要注意的是，修改 Hosts 仅可以针对域名被屏蔽而 IP 地址未被屏蔽的情况，若是 IP 地址被屏蔽，修改 Hosts 会毫无作用。本软件也提供清理已失效的 IP 的功能，让你的手机 Hosts 也保持干净。
★清理缓存文件
过多的缓存文件将极大的浪费系统内的空间，并拖慢系统的运行，缓存文件大多由浏览网页，软件内置的广告等带来，清除它们将可以得到更多的空间。本软件可以在短时间内将手机内的缓存文件全部理清干净，不留下任何残余。
★Root 和 Busybox
支持 Root 检查和 busybox 更新安装。特别的，针对 CM 的 4.0 系统（包括由其衍生的 MIUI 等），也提供了相关的解决方案。
★杂项功能
杂项功能大多都是辅助类的，例如媒体扫描，Dalvik 清理等。本软件设置项也非常丰富，请自行使用体会。
本软件特有的适配度功能，可以清楚的指示出当前设备是否与本软件合拍。其指示数从 1 开始，表示完全不匹配，到 9 结束，表示完全匹配。匹配度是通过判断手机型号，系统版本，ROM 等等综合打分，匹配度过低可能会引起功能无效，本软件卡顿感强烈，或是界面不能完全显示等问题。
