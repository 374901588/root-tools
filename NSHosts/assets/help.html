NSHosts  (ver 1.0.0)

★软件介绍
这是一款用于 Android 系统的 Hosts 修改工具，它可以方便的对您的 Hosts 文件进行管理或是搜索 IP 地址，使您可以正常访问一些域名被封的站点。本软件界面友好，可以很快上手。
★运行条件
要正常使用本软件，您的手机需要有 root 权限（正确安装了 busybox 的情况更佳），这在一般的第三方 ROM 中，都已集成，例如 HiApk, GoApk, 机锋，MIUI 等发布的 ROM，均包含本软件所需的条件。本软件需要运行在 Anroid 2.2 或更高版本，对于 G6 / G7 或是一些特定的机型，使用本软件需要将手机解为 S-OFF 状态，新款手机如 EVO 3D 等，使用官方解锁则不需要 S-OFF。
★使用方法
1) 进入本软件后，将看到四个功能模块，分别是修改 Hosts，搜索 IP，选项设置和使用说明。
2) 修改 Hosts 功能允许您直接对 Hosts 进行编辑，包括添加，删除，去除已失效的 IP，或是手动编辑。编辑完成后，可以将当前的内容保存到 /etc/hosts，设备重启后即生效。
3) 搜索 IP 功能允许您搜索一个域名所对应的 IP 地址，也可以将搜索结果有选择性的添加到 Hosts 内。已经包含了一些常用的站点供选择，省去了每次输入网址的麻烦。
4) 选项设置允许您设置 Name Server，设置是否搜索后默认选中，也可以更新常用站点列表。您可以将 Name Server 设置为您所信任的服务器，预设为 Google 的 DNS 服务器，即 8.8.8.8
